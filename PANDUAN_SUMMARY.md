## ğŸ”‘ Poin-Poin Penting

### Setup (Pertama Kali - ~10 menit)
```bash
1. npm install              # Install dependencies
2. Buat .env.local         # Konfigurasi environment
3. Buat database PostgreSQL # CREATE DATABASE
4. npm run dev             # Jalankan server
5. http://localhost:3000   # Akses browser
```

### Development (Setiap Hari - ~30 detik)
```bash
1. npm run dev             # 1 command saja
2. Browser otomatis reload # Hot reload
3. Ctrl+C untuk stop       # Ketika selesai
```

### Troubleshooting
```bash
1. Lihat error di terminal dan console (F12)
2. Cari di TROUBLESHOOTING_LOCALHOST.md
3. Ikuti solusi
4. Restart jika perlu
```
